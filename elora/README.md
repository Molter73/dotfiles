## Configure udev

Follow these instructions to create udev rules that will let you run
Vial without root permissions:
    https://get.vial.today/manual/linux-udev.html

## QMK keycodes

When editing keys in Vial, double clicking a key lets you input QMK
code directly, QMK codes can be found here:
    latest: https://github.com/qmk/qmk_firmware/blob/master/docs/keycodes.md
    permalink: https://github.com/qmk/qmk_firmware/blob/faf77f1651f7660c55f7a8f2a7b653808a29e0e3/docs/keycodes.md

## XF86 events
For special key events like play, pause, brightness up/down, etc..
The identifiers for mapping in things like sway can be found here:
    https://wiki.linuxquestions.org/wiki/XF86_keyboard_symbols
    https://cgit.freedesktop.org/xorg/proto/x11proto/tree/XF86keysym.h
